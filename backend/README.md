# Система мониторинга АРМ (Бэкенд)

### Описание

Бэкенд-часть проекта, разработанная на Python с использованием фреймворка Flask. Она отвечает за считывание данных о вычислительных ресурсах из файла `monitoring_data.xlsx` и предоставление их в формате JSON через REST API.

### Используемые технологии

- **Python**: Основной язык программирования.
- **Flask**: Легковесный веб-фреймворк для создания API.
- **Pandas**: Библиотека для работы с данными, используется для чтения файлов Excel.
- **Openpyxl**: Движок, необходимый для pandas для работы с файлами `.xlsx`.
- **Flask-CORS**: Расширение для решения проблем с Cross-Origin Resource Sharing (CORS) при взаимодействии с фронтендом.

### Установка и запуск

1.  Убедитесь, что у вас установлен Python.
2.  Перейдите в папку `backend/` в терминале.
3.  Установите необходимые библиотеки, используя файл `requirements.txt`:
    ```bash
    pip install -r requirements.txt
    ```
4.  Убедитесь, что в этой же папке находится файл `monitoring_data.xlsx` с актуальными данными.
5.  Запустите сервер:
    `bash
python app.py
`
    Сервер будет доступен по адресу `http://localhost:5000`. API-эндпоинт для получения данных — `http://localhost:5000/api/data`.
